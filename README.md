# SmartParkingApp-EzPark (UTM Capstone Project 2018)
EzPark is developed to help users to find available parking spot in Malaysia.

# Additional Features
+ Find your car using GPS
+ SMS-payment
+ Alert Notification when parking time expires
+ Report double park
